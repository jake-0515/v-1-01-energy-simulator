🤖 1단계: 컴퓨터 비전과 MediaPipe

👁️ 컴퓨터 비전(Computer Vision)이란?
사람이 눈으로 보고 뇌로 인지하듯이, 컴퓨터가 카메라나 이미지를 통해 세상을 시각적으로 분석하고 이해하게 만드는 인공지능 기술 분야입니다.
🖐️ 구글 미디어파이프(MediaPipe) 손가락 추적

아래는 ChatGPT가 추천하는 가장 정석적인 웹캠 연동 미디어파이프 기본 예제입니다. 복사해서 test.py에 덮어씌워 보세요.

python (AI Hand Tracking) 
import cv2
import mediapipe as mp

# 1. 미디어파이프 솔루션(시각화 도구 및 손 인식 모듈) 불러오기
mp_hands = mp.solutions.hands
mp_drawing = mp.solutions.drawing_utils

# 2. 손 인식 모델 옵션 설정 (최대 1개의 손 인식, 최소 신뢰도 0.5)
hands = mp_hands.Hands(
    max_num_hands=1,
    min_detection_confidence=0.5,
    min_tracking_confidence=0.5
)

# 3. 컴퓨터의 0번 웹캠 켜기
cap = cv2.VideoCapture(0)

while cap.isOpened():
    success, image = cap.read()
    if not success:
        continue

    # 미디어파이프는 RGB 포맷을 사용하므로 BGR 이미지를 변환합니다.
    image_rgb = cv2.cvtColor(image, cv2.COLOR_BGR2RGB)
    
    # 이미지에서 손 관절 위치 탐색 프로세스 진행
    results = hands.process(image_rgb)

    # 손이 감지되면 21개 관절 위에 뼈대 그리기
    if results.multi_hand_landmarks:
        for hand_landmarks in results.multi_hand_landmarks:
            mp_drawing.draw_landmarks(
                image, 
                hand_landmarks, 
                mp_hands.HAND_CONNECTIONS
            )

    # 결과 화면 창에 띄우기
    cv2.imshow('AI MediaPipe Hands', image)

    # 'q' 키를 누르면 루프 탈출 및 프로그램 안전 종료
    if cv2.waitKey(1) & 0xFF == ord('q'):
        break

cap.release()
cv2.destroyAllWindows()
