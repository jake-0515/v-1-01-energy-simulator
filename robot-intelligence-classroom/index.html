01. 인공지능의 범위와 진화

우리는 인공지능(AI) 시대에 살고 있어요. 하지만 인공지능, 머신러닝, 딥러닝은 모두 같은 단어일까요? 어떤 차이가 있는지 알아봅시다.

인공지능의 3가지 핵심 범위

💻 인공지능 (Artificial Intelligence)

인간의 지능(생각, 학습, 판단)을 컴퓨터 프로그램으로 흉내 내는 모든 기술을 통틀어 부르는 넓은 개념이에요.

📊 머신러닝 (Machine Learning)

인공지능의 한 종류로, 사람이 일일이 규칙을 코딩하지 않고 데이터를 주면 컴퓨터가 스스로 규칙을 찾아 학습하는 기술이에요.

🧠 딥러닝 (Deep Learning)

머신러닝의 하위 기술로, 인간 뇌의 신경세포(뉴런) 구조를 본뜬 인공신경망을 이용해 컴퓨터가 스스로 특징을 찾아내며 심화 학습하는 기술이에요.

인공지능의 두 얼굴: 약인공지능 vs 강인공지능

● 약인공지능 (Narrow AI)
  • 주어진 특정 작업 한 가지만 수행할 수 있어요.
  • 현재 우리가 마주하는 거의 모든 AI가 여기에 속해요.
  • 예: 알파고(바둑), 파파고(번역), 가위바위보 분류기.
★ 강인공지능 (General AI)
  • 인간처럼 스스로 생각하고 계획하며 여러 일을 해내요.
  • 감정과 의식을 가질 수 있는 자아 성찰형 인공지능이에요.
  • 예: 영화 속 자비스, 도라에몽 (아직 연구 단계).

02. 머신러닝의 학습 공식: T-E-P

인공지능 학자 톰 미첼(Tom Mitchell) 교수는 머신러닝을 세 가지 요소로 설명했어요. 농구 자유투 연습을 통해 이 공식을 알아봅시다!

🏀 농구 훈련으로 알아보는 T-E-P 공식
  • T (Task - 작업): 해결하고자 하는 문제 ➔ 농구공 골대에 넣기
  • E (Experience - 경험): 컴퓨터가 학습하는 데이터 ➔ 반복해서 공을 던진 연습 횟수
  • P (Performance - 성능): 얼마나 잘해졌는지 측정하는 기준 ➔ 슛 성공률(%)
현재 성능 P (슛 성공률) 0%

특징 추출 (Feature Extraction)

컴퓨터는 강아지 사진을 통째로 이해하지 못해요. 그림 속에서 분류에 중요한 '특징(Feature)'을 뽑아내 수치 벡터로 변환합니다. 강아지를 스캔해 봅시다!

🔎 스캐너 레이저 점 클릭하기
강아지 머리, 몸체에 찍힌 노란색 조준점을 클릭하면 센서가 특징을 숫자로 스캔합니다.
🤖 추출된 특징 벡터 (Feature Vector)
[ ? , ? , ? ]

귀의 둥글기(0~1), 털의 밝기(0~2), 추정 무게(kg)가 숫자로 정렬되어 AI 모델의 최종 입력값이 됩니다.

03. 지도학습 (Supervised Learning)

지도학습은 문제(데이터)와 정답(레이블)을 사람이 직접 주고서 공부시키는 방식이에요.

✔ 분류 (Classification)

데이터를 서로 다른 그룹(카테고리)으로 쪼개는 작업이에요.

1. KNN (K-최근접 이웃)

"가장 가까운 곳에 있는 K개의 이웃이 뭐지?" 주변 이웃과 동일한 클래스로 판정해요. (예: 주위 다수가 고양이면 나도 고양이)

2. SVM (서포트 벡터 머신)

두 그룹 사이에 가장 넓은 빈 도로(경계)를 건설하여 안전하게 데이터를 분류하는 알고리즘이에요.

3. 의사결정나무 (Decision Tree)

"귀가 뾰족한가? ➔ 예/아니오" 처럼 스무고개 질문을 던져 리프 노드까지 타고 가 데이터를 분류해요.

✔ 회귀 (Regression)

끊기지 않고 이어지는 연속적인 수치(숫자)를 예측하는 작업이에요.

실생활 예시: 공부한 시간 ➔ 시험 점수 예측, 스마트폰 충전량 ➔ 남은 사용 시간(분) 예측, 기온 ➔ 아이스크림 판매량 예측.

비지도학습 (Unsupervised Learning)

비지도학습은 정답(레이블)이 없는 데이터만 던져주고 컴퓨터가 스스로 공통점과 차이점을 찾도록 하는 학습법이에요.

✔ 군집화 (Clustering)

데이터들의 성격이나 생김새를 보고 공통된 것끼리 모둠(그룹)으로 묶는 기술이에요.

실생활 예시: 넷플릭스 영화 추천 (내가 시청한 영상들의 패턴을 분석해 비슷한 영화 마니아 그룹에 날 묶어서 추천)

📐 군집의 완성도 평가 기준
  • 응집도 (Cohesion): 같은 그룹에 속한 친구들끼리 얼마나 가깝게 모여 있는가? (높을수록 좋음)
  • 분리도 (Separation): 서로 다른 그룹끼리 얼마나 멀리 벌어져 있는가? (높을수록 좋음)
✔ 연관 규칙 (Association Rule)

대량의 데이터 분석을 통해 항목 간의 "A를 사면 B도 같이 사더라"와 같은 흥미로운 규칙을 찾는 기술이에요.

실생활 예시: 기저귀를 산 손님이 밤에 시원한 맥주를 장바구니에 같이 담는 행동 분석 규칙, 유튜브에서 '라면 끓이기'를 본 사람에게 '김치 리뷰' 영상을 연관 동영상으로 추천하기.

04. 생물학적 뉴런 vs 인공 뉴런

인공지능의 딥러닝은 우리 머릿속 뇌신경세포(뉴런)의 신호 전달 방식을 모델링하여 만들어졌어요. 둘의 짝꿍이 어떻게 매핑되는지 클릭해 봐요!

가지돌기 (Dendrite)

➔ 입력값 (Inputs: x1, x2, x3)

외부 자극이나 다른 세포로부터 신호를 전달받는 빨대 역할을 해요.

시냅스 (Synapse)

➔ 가중치 (Weights: w1, w2, w3)

자극이 지나가는 통로로, 통로가 넓을수록 신호가 더 강하게(가중치 높게) 흐릅니다.

세포체 (Soma / Cell Body)

➔ 가중합 & 활성화 함수

들어온 신호를 모아 연산하고, 일정 수치 이상의 자극이 차면 신호를 전달할지 정해요.

축삭돌기 (Axon)

➔ 출력값 (Output: y)

결정된 최종 판단(전기 자극)을 다음 뉴런으로 내보내는 통로입니다.

🧠 뉴런 세포 카드 클릭
상단의 생물학적 뉴런 카드를 클릭하면 해당하는 인공신경망의 수학적 수학 연산 매칭 원리가 안내됩니다.
