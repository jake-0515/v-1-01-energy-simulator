📊 1단계: 정형/비정형 데이터와 전처리

인공지능 학습을 위해 컴퓨터가 이해할 수 있도록 데이터를 가공하는 과정을 알아봅시다.

📂 정형(Structured) vs 비정형(Unstructured) 데이터
* **정형 데이터**: 엑셀 표나 데이터베이스처럼 행과 열이 정해진 규격화된 데이터 (예: 붓꽃 데이터).
* **비정형 데이터**: 텍스트, 이미지, 소리, 비디오처럼 형태가 일정하지 않고 연산이 바로 불가능한 데이터 (예: 강아지/고양이 사진).

🏫 실생활 비유로 이해하는 데이터 전처리(Rescaling)

만약 철수가 수학 시험에서 90점을 받고, 10점 만점인 미니 퀴즈에서 6점을 받았다면 어떤 점수가 더 훌륭할까요? 단순히 숫자만 보면 90이 훨씬 커 보이지만, 만점 기준과 점수 분포가 다르기 때문에 직접 비교하기 어렵습니다.

이처럼 데이터의 범위가 다를 때 점수를 0 ~ 1 사이로 통일하거나(정규화), 평균을 0으로 맞추어(표준화) 공평하게 맞춰주는 작업을 **데이터 전처리**라고 합니다.

원시 데이터: 가공되지 않은 학생들의 시험 원점수 (만점 기준 100점)

🍊 2단계: Orange3 설치 및 구성 요소

코딩 없이 마우스 드래그로 머신러닝을 구현할 수 있는 도구인 오렌지3를 설치해봅시다.

💻 Orange3 데이터 마이닝 프로그램이란?
구글, 네이버 등 대형 IT 플랫폼들도 사용하는 머신러닝 파이프라인 구축 프로그램입니다. 위젯(기능 블록)을 드래그하여 실시간으로 데이터 마이닝, 시각화, 인공지능 분석을 수행할 수 있습니다.

🔽 다운로드 안내:

  • [Orange3 공식 웹 다운로드 페이지] 로 접속합니다.
  • 설치 도중 Miniconda3를 같이 설치하는 창이 뜨면 확인을 누르고 모두 설치해 줍니다.
  • 설치 완료 후 실행하면 아래와 같이 데이터 전처리, 시각화, 평가 위젯들이 모인 대시보드가 열립니다.

🧩 핵심 위젯 분류표:

Data 데이터를 불러오고(File), 테이블로 열어보는(Data Table) 위젯군
Visualize 데이터 분포를 그리는(Scatter Plot, Box Plot) 시각화 위젯군
Transform 데이터의 범위를 변경하거나 학습용/테스트용으로 섞어 나누는 위젯군
Model & Evaluate 기계학습 알고리즘(SVM, KNN, Logistic Regression) 및 평가 위젯군

🧩 3단계: Orange3 붓꽃(Iris) 분석 시뮬레이터

우측 워크스페이스에서 마우스 드래그로 위젯을 연결하여 실제 데이터 마이닝 과정을 미리 체험해봅시다.

🎮 드래그 & 연결 실습 방법:
1. **`File`** 위젯 우측에 있는 주황색 연결 포트(●)를 마우스 왼쪽 버튼으로 클릭 후 드래그합니다.
2. 점선이 나오면, 연결하고자 하는 대상 위젯(**`Data Table`**, **`Scatter Plot`**, **`Data Sampler`**)의 왼쪽 회색 포트(●)에 마우스를 뗍니다.
3. 연결선이 완성되면 위젯이 환하게 활성화됩니다. 활성화된 위젯을 클릭(터치)하시면 팝업창으로 데이터의 내용이 시각화되어 출력됩니다!

👉 File 위젯은 기본적으로 150개의 붓꽃 표본 데이터를 가집니다. 위젯들을 모두 연결하여 데이터 표와 그래프, 그리고 70% 학습 데이터 분할 정보를 직접 열람해 보세요!

🐈 4단계: 개와 고양이 이미지 분류 (임베딩 체험)

비정형 데이터인 사진을 인공지능이 어떻게 숫자로 변환하고 분류하는지 체험해 봅시다.

🤖 이미지 임베딩 (Image Embedding) 이란?
컴퓨터는 고양이나 강아지 사진을 픽셀 그대로 분석하여 지도학습하기 어렵습니다. 구글의 합성곱신경망(CNN) 기반 모델이 이미지에서 털의 재질, 눈과 귀의 모양, 주둥이 비율 등을 분석하여 수천 개의 숫자 정보(특성 벡터, Feature Vector)로 가공해주는 기법입니다.
